# Student Management System

A full-stack Student Management System developed using Spring Boot, HTML, CSS, JavaScript, and SQL Server.

## Features

- Admin Dashboard
- Student Dashboard
- Student Registration & Login
- Course Management
- Fee Management
- Exam Management
- Certificate Verification
- Notifications System
- Expense Management
- PDF Certificate Generation
- Email Integration
- Stripe Payment Integration

## Technologies Used

### Backend
- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate

### Frontend
- HTML
- CSS
- JavaScript

### Database
- Microsoft SQL Server

## Tools & Platforms
- Git & GitHub
- Maven
- STS 
- Render (Deployment)
